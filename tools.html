Command line tools and utilities¶

A couple of command line tools and utilities help you working with RISC-V and can serve as entry point for extending this package.

Discover RISC-V ISA Variants: riscv-describe-isa-variant¶

Describe a RISC-V ISA variant in a human-readable way.

usage: riscv-describe-isa-variant [-h] variant

Positional Arguments¶

variant

Variant to describe, example: RV64GC

Usage Example:

$ riscv-describe-isa-variant RV64GC
RV64GC
XLEN=64, 32 integer registers (I)
Extensions:
    A        Atomics
    C        16-bit Compressed Instructions
    D        Double-Precision Floating-Point
    F        Single-Precision Floating-Point
    M        Integer Multiplication and Division
    Zicsr    Control and Status Register Access
    Zifencei Instruction-Fetch Fence

Generate a random instruction stream: riscv-random-asm¶

Generate a stream of random RISC-V instructions. Those sequences are valid instruction but not necessarily valid programs, in particular they don’t have actual control flow.

Generate sequence of assembler instructions.

usage: riscv-random-asm [-h] [-v VARIANT] [-i INSTRUCTION] [--version] [N]

Positional Arguments¶

N

Number of assembler instructions

Default: 10

Named Arguments¶

-v, --variant

Restrict to variant

Default: “RV32I”

-i, --instruction

Restrict to instructions

--version

Display version

Usage Example to generate 2 random RV32I instructions:

$ riscv-random-asm -v RV32I 2
sb x30, 943(x1)
sra x0, x7, x7

Or restrict to one particular instruction:

$ riscv-random-asm -i or 5
or x0, x12, x11
or x13, x18, x0
or x8, x10, x30
or x2, x4, x2
or x0, x2, x27
